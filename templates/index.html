Fase 2

Fundamentos del modelo

Qué es la regresión logística y qué ocurre por detrás

¿Qué hace el modelo?

La regresión logística es un modelo de clasificación binaria. En lugar de predecir un número continuo, estima la probabilidad de que una observación pertenezca a la clase positiva (por ejemplo, aprobar = 1).

P(Y=1 | X) = 1 / (1 + e−(β₀ + β₁x₁ + β₂x₂ + …))
  • β₀ — intercepto del modelo
  • βᵢ — peso de cada variable
  • Sigmoide — convierte el score lineal en probabilidad (0 a 1)
  • Si P ≥ 0.5 → clase 1; si P < 0.5 → clase 0

Función sigmoide

Detrás del modelo, scikit-learn optimiza los coeficientes β para minimizar la función de pérdida logística.
